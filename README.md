# SyncCraft

A modern **web-based project management application** built with **Next.js** that enables teams to manage projects, track tasks, and collaborate.

This project focuses on **speed, security, and real-time collaboration**, making it ideal for small to medium-sized teams who want a clean and efficient workflow.

---

## 🚀 Tech Stack

* **Frontend & Backend**: Next.js (App Router)
* **Database & Auth**: Supabase
* **Authentication**: Supabase Auth + Bcrypt for password hashing
* **Real-Time Features**: Supabase Realtime
* **Styling**: CSS / Tailwind CSS (if applicable)
* **Deployment**: Vercel

> ⚠️ This project is **web-only** and not intended for mobile applications yet.

---

## ✨ Features

### 👤 User Management

* Email & password authentication
* Secure password hashing using bcrypt
* User roles:

  * Admin
  * Project Manager
  * Team Member

### 📁 Project Management

* Create and manage multiple projects
* Add team members to projects
* View project-level task progress

### 📝 Task Management

* Create tasks with:

  * Title
  * Description
  * Priority
  * Start & due dates
  * Points
* Assign tasks to team members
* Task statuses:

  * To Do
  * In Progress
  * Done

### 📊 Progress Tracking

* Project completion overview
* Task count by status
* Bar chart, Pie Chart & Gantt Chart for better visualizaion

### 🔐 Security

* Role-based access control
* Encrypted data storage
* Secure authentication flow
